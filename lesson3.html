Securing Your Network:

Anyone can use a private network if it is not properly secured. Wireless networks require an access point which any device can access within a certain range which can include up to several hundred feet. As a first step in protecting a private network, limit the transmitter range to your house or business. Also, change the default name and password of your router. Set up anti-virus and anti-spyware software in addition to a firewall. Each device has a unique Media Access Control address; set your network to only allow specific addresses. Keep your Service Set Identifier broadcast turned off. While all these things help protect your network, encryption is the most reliable protection. Encryption disarranges information sent via the internet to the point of unreadability. Lastly, turn off your network when you are not using it.
