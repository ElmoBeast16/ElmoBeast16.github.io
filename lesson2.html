Public Wifi:

Public Wi-Fi can be risky to use, however, there are measures you can take to protect your devices on public Wi-Fi. If you are at a coffee shop, library, hotel, or the like, be sure to verify the name of the network. Before connecting, turn off file sharing. If you have windows, identify the network as public. Setting up a Virtual Private Network makes it harder for people to get your files by intercepting and enciphering all information exchanged between the server and your device. When setting up accounts online, use different passwords for each one and set up two-factor authentication if possible. When online, do not assume any website or app is safe. Do not access things like banking accounts on a public network. When done using public Wi-Fi, log out of any service you were using and set your device to forget, or not connect automatically, to the network. Turn off Wi-Fi when you are not using it.
